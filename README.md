# Work in progress
# Description
3D runner is a mobile hyper casual game where the player has to avoid collisions and get the highest score.

# Control
- Move Left - "Swipe Left"  
- Move Right - "Swipe Right"
- Jump - "Swipe Up"
- Fall Down - "Swipe Down"
